## Gym time API


# Features

1-Login

2-add gym

3-Update gym information

4-Delete gym

5-View all gyms

6-Search gym by name

7-add comment

8-delete comment 

9-add rating 

10-delete unrelated comments



# Stack

Runtime Environment: `Node.js`

Language: `TypeScript`

Web Server: `fasify`

ORM: `mongoose`

DB: `mongo`
